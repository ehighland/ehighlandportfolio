Code

I have a variety of projects on GitHub. You can view them right here. Some repositories contain more recent personal projects and some contain assignments from my undergrad classes.

My personal projects include useful and informative sites, such as this very site you are on right now as well as my wedding website. Of course, useful and informative isn't always fun and I like to mix it up. For example, I love horror movies and I made a website for a short lived horror project. I made a shuffle feature to select a horror movie to watch every day for the month of October, essentially trying to code away my indescision. Things fell apart when I decided I should be watching and writing a review of each movie every day. It was a cool idea, except that I had never written a review before. That said, it was fun while it lasted and I did watch a lot of new movies.

My school project I am most proud of is a hi-fi mockup of my idea for a redesign for Mendley desktop. I had worked on this project with another student for the Human-Computer Interface Design class (COMP 441) at Loyola University Chicago. I really enjoyed learning about making an interface intuitive for the user.
